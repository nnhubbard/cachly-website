# Cachly Website

The official website for [Cachly](https://www.cachly.com) - Premium Geocaching for iOS.

## About Cachly

Cachly is a premium iOS geocaching application and an Authorized Developer with Geocaching HQ. The app provides a comprehensive geocaching experience across iPhone, iPad, Apple Watch, and CarPlay.

- **App Store Rating:** 4.8 stars (3,500+ ratings)
- **Languages:** 17 supported languages
- **Developer:** Zed Said Studio LLC

## Key Features

- **Pro Offline Maps** - OpenStreetMap-based vector maps for offline use worldwide
- **CarPlay Integration** - Full geocaching support on car dashboards
- **Apple Watch App** - Search, navigate, and log caches from your wrist
- **Live Search Filtering** - 63 different filter types for finding caches
- **Offline Lists** - Save unlimited caches for offline access
- **DeLorme Grid** - Track geocaching progress across US states
- **TB Scanner** - QR code scanner for trackable items
- **GPX Export** - Export cache data for use in other apps
- **Translation** - Translate logs and descriptions to multiple languages
- **Counties Tracking** - Monitor progress across counties
